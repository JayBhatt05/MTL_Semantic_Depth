# MTL_Semantic_Depth
Domain Generalized Multi-Task Learning for Semantic Segmentation and Depth Estimation Tasks  
Datasets Used: CityScapes ( https://www.cityscapes-dataset.com/ ), SynScapes ( https://synscapes.on.liu.se/ )  
  
For Semantic Segmentation, segmentation map label IDs are converted to their respective train IDs using the labelling scheme given at https://github.com/mcordts/cityscapesScripts/blob/master/cityscapesscripts/helpers/labels.py
